Программа создавалась на Python 3.8.8.
Перед началом работы в командной строке введите:
pip install speechrecognition (для распознования текста)
Затем (если вы используете Python 3.8) введите:
pip install PyAudio-0.2.11-cp38-cp38-win_amd64.whl
Если возникнут проблемы, обратитесь к Google: как установить библиотеку PyAudio на Windows.

Дальше:
pip install textblob (для перевода)

Перед запускам translate.py запустите test.py. Найдите нужное устройство, запомните его индекс и введите в нужную строку в
translate.py.
